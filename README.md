# Chakra Python Quest

**Learn Python Through the 7 Chakras of Wisdom**

A visual Python learning game themed around Indian spirituality. Journey through the 7 Chakras — from Muladhara (Root) to Sahasrara (Crown) — to master Python from beginner to advanced.

## About

Each chakra represents a Python skill level with teachings and quiz questions. Answer correctly to earn **Karma Points**, awaken chakras, and unlock the next level. The game features animated mandalas, lotus flowers, particle effects, and spiritual mantras from the Bhagavad Gita.

## The 7 Chakra Levels

| # | Chakra | Symbol | Python Topic | Difficulty |
|---|--------|--------|-------------|------------|
| 1 | **Muladhara** (Root) | LAM | Variables, Print, Input, Comments | Beginner |
| 2 | **Svadhisthana** (Sacral) | VAM | Strings, Numbers, Booleans, Operators | Beginner |
| 3 | **Manipura** (Solar Plexus) | RAM | If/Else, For, While, Break/Continue | Intermediate |
| 4 | **Anahata** (Heart) | YAM | def, return, args, lambda | Intermediate |
| 5 | **Vishuddha** (Throat) | HAM | Lists, Dicts, Sets, Tuples | Intermediate |
| 6 | **Ajna** (Third Eye) | OM | Classes, Inheritance, Polymorphism | Advanced |
| 7 | **Sahasrara** (Crown) | AUM | Decorators, Generators, Comprehensions | Advanced |

## Installation

### Prerequisites
- Python 3.7 or higher
- pip (Python package manager)

### Setup

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/Python-game.git
cd Python-game

# Install dependencies
pip install pygame

# Run the game
python chakra_python_game.py
```

## How to Play

1. **Begin Journey** — Start from the first chakra (Muladhara)
2. **Read the Teaching** — Each question shows a concept explanation first
3. **Answer the Quiz** — Select the correct answer to earn Karma Points
4. **Awaken Chakras** — Complete all 5 questions to unlock the next chakra
5. **Reach Enlightenment** — Awaken all 7 chakras to master Python!

### Controls

| Key | Action |
|-----|--------|
| **Mouse Click** | Navigate menus, select answers |
| **A / B / C / D** | Quick answer selection |
| **1 / 2 / 3 / 4** | Quick answer selection |
| **Enter / Space** | Proceed to next screen |
| **Escape** | Go back |

## Features

- 35 curated Python questions (5 per chakra)
- Teaching screen before each question explains the concept
- Animated mandalas, lotus flowers, and OM symbol
- Twinkling star background with particle effects
- Karma Points scoring system
- Auto-save progress to continue later
